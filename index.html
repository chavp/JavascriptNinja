สวัสดีครับ นิน นิน

งานเขียนในนี้ของผม จะพยายามรวบรวมเทคนิค Javascript ของนินจา เอาไว้ให้ได้มากที่สุด

แรงบรรดาลใจของผมในภาษา Javascript นี้เริ่มมาจาก ผมเริ่มเขียนภาษาจาก VB6 -> C++ -> JAVA -> C# ไม่รู้จัก javascript เลย จนกระทั้งผมได้เขียนมันเมื่อไม่นานมานี้ พอเขียนมันมากเข้า มากเข้า เราก็เริ่มหลงไหลมันมากขึ้นเรื่อยๆแล้วละสิ

ไม่ได้ละต้องค้นคว้าจริงจังกันไปเลยจึงเกิดเป็นงานเขียน ชิ้นนี้ขึ้นมา

หัวข้อที่รวมไว้ตอนนี้ ก็มีดังนี้

  • library(อาวุธ)Javascript ที่นินจาจะต้องสนใจ
  • จะเป็นนินจาที่เก่งกาจได้ต้องเรียนรู้ javascript ขั้นสูง
  • การทดสอบ javascript ของนินจา
  • ความสามารของ javascript ที่ทำให้นินจา cross-browser(เก่งได้ทุกสภาวะ)
